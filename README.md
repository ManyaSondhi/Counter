# React SPA – Event Handling Demo

This project is a React Single Page Application (SPA) built using Vite. It demonstrates core React concepts such as component-based architecture, state management, and event handling. The application includes a Counter, Form, To-Do List, and Theme Toggle module.

All interactions update dynamically without page reload using React Synthetic Events.

## Features
- Single Page Application behavior
- Fast development using Vite
- Real-time UI updates
- Counter with increment and decrement
- Form input handling and submission
- To-Do list with add and delete functionality
- Light and Dark mode toggle
- Clean and reusable components

## Events Used
- onClick – Button interactions
- onChange – Input handling
- onSubmit – Form submission handling

## Technologies Used
- React.js
- Vite
- JavaScript (ES6)
- HTML5
- CSS3

## How to Run the Project
1. Install Node.js  
2. Clone the repository  
3. Open terminal in project folder  
4. Run `npm install`  
5. Run `cd SPA`  
6. Run `npm run dev`  
7. Open browser and visit:  
   `http://localhost:5173/`
